# TVBox 配置仓库

## 文件结构
- `tvbox.json` - 主配置文件
- `lib/` - 所有依赖文件（jar、js、txt、m3u）

## 访问地址
- jsdelivr CDN: `https://cdn.jsdelivr.net/gh/liuzzh1971/tvbox-config@main/tvbox.json`
- GitHub raw: `https://raw.githubusercontent.com/liuzzh1971/tvbox-config/main/tvbox.json`

## PeekPili 使用方法
在 PeekPili 中添加配置地址:
```
https://ghproxy.net/https://raw.githubusercontent.com/liuzzh1971/tvbox-config/main/tvbox.json
```

或直接用 jsdelivr（不走代理）:
```
https://cdn.jsdelivr.net/gh/liuzzh1971/tvbox-config@main/tvbox.json
```
